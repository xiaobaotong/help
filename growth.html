# 🚀 营销指南

# 前言

每一个创作者，都要开始把自己的价值变成可交付的产品，并卖出去。这样才能让你的创作更加持续，也能和更多人建立长期深入的关系。 所以在积累了一定量的内容后，也需要把小报童推荐给朋友。

不要有心虚的感觉，责任往往是一个人成长的最佳助力。

# 首发实践

首发是非常重要的宣传手段，这里推荐 Lightory (opens new window) 的首发策略

# 1. 写清楚利益点，以及附上一些案例

站在 Small Businesses Owner 的视角，写一些关于产品、创业和投资的观点。正如这个公众号过去一年的文章。
区别在于，公众号写文终究是一件过于正式的事情，把一个想法发展成完整文章也是一个痛苦过程——我从来是厌恶形式的。所以过去我只是偶尔一写。
而在这个专栏，我会更随意地记录想法，每篇两、三百字，精炼地表达。专栏里的每篇内容，都是公众号文章的 Prototype。
这换来的是，更新频度大幅提升。如图所示，我尝试写了两周，写了 8 篇——这已是一个克制后的结果。

# 2. 控制订阅者预期，明确边界：

一、没有社群，没有圈子，没有连接。你不会因为订阅这个专栏而获得人脉。
二、只是零碎的观察和记录，不成体系的输出。你不会立刻获得可落地执行的方法论。
三、这是一个完全兴趣导向的专栏，输出内容完全取决于我的兴趣所在。连我也不能预测下个月会写些什么。可能会是 Web3？\

# 3. 写清楚价格和放上订阅入口：

不要害羞，写上价格，对自己的内容有信心。
记得用图片二维码做入口，比阅读全文效果好很多。

# 首发数据参考

BuildForever (opens new window) 的东炜黄同学在他的小报童分享了一组冷启动数据，这里摘录部分如下，对于还没进行推广的创作者可以参考：

  • BuildForever 微信公众号和 newsletter（Revue 平台） 共约 3500 位朋友关注。
  • 开通小报童专栏后，我发了 2 篇公众号文章（mp.weixin.qq.com 和 mp.weixin.qq.com） 、1 封停更 Revue 邮件。
  • 若按公众号和 newsletter 的合计阅读量（约 1100）来估算，阅读转化率为 11.5%。
  • 截至此时，共计 127 位朋友付费支持。
  • 算下来，付费转化率约 3.5%。
  • 参考 Substack 提供的数据 5~10%（ substack.com[2]），偏低了。
  • 但已破百，知足。

对于这次首发，他的反思是：大约 70% 的订阅量来自首发文章。如果当时冷静一些，好好打磨，不由着刚刚开通专栏的兴奋急于昭告天下，或许会再多一些阅读和分享，多一些订阅。

不过根据我们的经验和数据，经过长期认真的运营，后续转化率会持续提高的，首发重要，但并非决定生死。如他所说：数据归数据。订阅始终是一种约定关系，是你我之间的交流窗口。「we don’t stop being human when we go online.」

# 长期营销实践

除了首发，如果希望专栏常青，那么还需要有长期营销的手段，而最朴素的手段就是口口相传。所以请务必保持专栏的质量，然后给予时间，就一定会带来更多的读者。

当然，在确保上述事项都能做到之后，我们可以做些事情，稍微让其加速，更加茁长的成长：

# 1. 日常更新中添加小报童

如果你日常有免费更新的内容，那么可以在其中提及小报童的订阅。

然阿姨 (opens new window)的建议是：

  1. 保证曝光：每次公号发文都会尽量带上小报童的专栏介绍，保障曝光率。
  2. 告知缘由：尽量在文中告知读者你会在小报童写什么，以及自己的收获是什么，给忠实读者补充更多背景信息。

# 2. 在朋友圈和社群中推荐新发的内容

修饼和冰清比较喜欢运营朋友圈和群，当她们在小报童发了新的内容，会同步在朋友圈和微信群预告。

冰清 (opens new window)分享的经验是：

  1. 不断重复：这不是在强推，而是在告诉别人，第一你做的这件事情很有价值，第二你做的非常棒（才能一直做下去），第三你是认真的（不是玩玩的）。
  2. 注重预览：由于小报童有预览部分，在预览中应尽量把读者的注意力和好奇心调动起来。这样有未付费的用户点进去的时候，就不会有「也就这样嘛」的感觉，而是「我要看看究竟他会讲些什么」。
  3. 要有自信：推广专栏时应该是充满着自信的，而不是充满着「匮乏感」。所谓「匮乏感」是实际上认为自己不行，必须得要通过付费用户数量多寡来证明自己，那样结果一定是失败的。正确的方法是，自己真正的在创造价值，然后欣喜的告诉大家，得到的回报会远超付出的价值。

tips：发朋友圈时，可以把当天小报的主题内容归纳一下（但是最核心的点不要说出来），如果有的话加上一点反馈，留言啥的效果更好。

修饼 (opens new window)分享的经验是：

  1. 每天坚持在抖音分享自己做的视频，就会有很多人来加微信或者微信群。
  2. 由于每天都会有新人加自己的微信号，为了避免全是一排二维码，可以只保留最近一期小报童发的内容。
  3. 内容作为视频的补充或者延伸，对于社群和朋友圈转化会更好。

# 3. 内容扩写

Lightory 的经验是，将小报童日常发布的某些内容，扩写成为一篇完整的文章，并在免费渠道发布。这样可以让未订阅的用户看到你的思考，又能看到专栏的价值。
