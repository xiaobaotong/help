# 😃 读者使用指南

感谢使用小报童订阅以及支持你喜欢的专栏作者。这里是一份简要的使用说明，希望能给你更好地用户体验。

# 如何找到我订阅的专栏/小册

# 在桌面端阅读

直接访问 xiaobot.net (opens new window)

# 在微信中阅读

在微信中关注【小报童投递】服务号，或者扫描二维码关注。请放心关注，除了你订阅的作者更新，以及订阅状态变更，我们是不会乱对你推送内容的。

关注后点击菜单栏，即可进入小报童首页，看到你订阅的专栏。

# 在邮箱中阅读

在投递设置中，开启邮箱投递，即可在邮箱中直接阅读最新的内容。

# 如何设置投递频率及渠道

  • 点击右上角头像
  • 点击投递设置
  • 点击投递频次，可以切换每天/每周，投递时间可以自定义。
  • 如果希望同期投递到邮件，可以绑定下常用邮箱

# 如何续订

为避免冲动消费，当订阅专栏还有七天到期的时候，在专栏详情页面即可看到续订按钮。

同时还会有三次微信续订提醒，即到期前七日、三日以及过期后七日。

你也可以在订阅列表最下方找到已经过期的专栏进行续订。

# 如何退订

# 专栏退订

在小报童微信服务号内，直接输入「退订」，并附上你的订单截图。我们会在 24 小时内人工处理。

注意，你可以在订阅后 48 小时内，申请退款。所有申请我们会人工进行审核。连续退款三次，账号将会进入冷静期，届时将无法订阅新的专栏。

# 小册退订

由于小册是买断制，为保护创作者的权益，购买后不支持退款。

如果你还有更多想聊的，想说的，想问的，请加我们微信号聊聊：Plidezus
